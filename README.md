# Tripplounge

Tripplounge is a white-label trip page service for travel advisors.

## Purpose

This repository hosts the public website and trip pages for Tripplounge. The goal is to provide clean, branded, client-facing trip pages that travel advisors can share with their own clients.

## What this site will include

- A neutral homepage for the Tripplounge brand
- Reusable trip page templates
- Custom trip pages for advisors and their clients
- Optional branded links connected to custom domains

## Notes

This repository is separate from Elite Tripps so the Tripplounge brand can stay neutral and white-label friendly.
